# Установка
Образ большой, собирается не быстро, места на диске занимает, вместе с моделями, порядка 20-30Гб. 
## Если linux/mac и установлен Make:
```
make build && make run
```
## Если делаем вручную:
```
# Собираем образ
docker build -t music .
# Выпускаем гуся
docker run \
		--privileged \
		--gpus all \
		--net=host \
		--restart=always \
		-e HF_ENDPOINT=http://dev.amberra.ru:8090 \
		music:latest
```

# Как проверяем:
После запуска контейнера поднимается веб-интерфейс, доступный по адресу http://localhost:7860. В нем нужно инициализировать модель (большая кнопка), предварительно включив чекбокс о необходимости инициализации LLM
После инициализации (если не упадет) нужно попробовать сгенерировать любую инструментальную мелодию - достаточно если будет 30-60 секунд. Текст не нужен - речь именно об инструменталке.

Держим пальцы крестиком чтобы хватило памяти видюхи.
